# Multi-Layer Network Stress Testing Tool

A powerful network diagnostic and stress testing utility written in Python. This tool supports various methods across Layer 4 and Layer 7, utilizing multi-threading and proxy integration for comprehensive network analysis.

## ✨ Features

- **Layer 4 Attacks:** TCP, UDP, SYN, ICMP, and various Amplification methods (DNS, NTP, RDP).
- **Layer 7 Attacks:** HTTP GET, POST, HEAD, and specialized methods to bypass protections (Cloudflare, DDoS-Guard).
- **Game Server Testing:** Specialized payloads for Minecraft, FiveM, Source Engine, and more.
- **Proxy Support:** Integrated with `PyRoxy` for HTTP/S, SOCKS4, and SOCKS5 support.
- **Real-time Monitoring:** Tracks PPS (Packets Per Second) and BPS (Bytes Per Second).
