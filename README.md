# IOWNCODZ - Multi-Layer Network Stress Testing Tool

A powerful network diagnostic and stress testing utility developed for the **ddos** project. This tool supports various methods across Layer 4 and Layer 7, utilizing multi-threading and proxy integration for comprehensive network analysis.

## ✨ Features

- **Layer 4 Attacks:** TCP, UDP, SYN, ICMP, and various Amplification methods (DNS, NTP, RDP).
- **Layer 7 Attacks:** HTTP GET, POST, HEAD, and specialized methods (Cloudflare, DDoS-Guard bypass).
- **Game Server Testing:** Payloads for Minecraft, FiveM, Source Engine, etc.
- **Proxy Support:** Fully integrated with `PyRoxy` for HTTP/S, SOCKS4, and SOCKS5.
- **Monitoring:** Real-time PPS (Packets Per Second) and BPS (Bytes Per Second) stats.
